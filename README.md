# mollyennis.com

personal portfolio site. vanilla html/css/js, built with vite.

## setup

```
npm install
npm run dev
```

## build

```
npm run build
npm run preview
```

## structure

- `index.html` — single-page app shell
- `main.js` — navigation, project loading, interactive demos
- `style.css` — all styles, responsive + dark mode
- `public/projects/` — markdown files for each project (loaded at runtime)
- `public/media/` — images, videos, svgs
- `public/fonts/` — commit mono, cash sans
